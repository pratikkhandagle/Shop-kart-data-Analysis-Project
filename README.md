# Shop-kart-data-Analysis-Project

This project analyzes retail sales data to uncover key business insights related to revenue performance, customer behavior, product profitability, and loss-generating transactions. The objective of this project is to transform raw transactional data into meaningful insights that help businesses make data-driven decisions.

The analysis was performed using SQL for data exploration and analysis and Power BI for building interactive dashboards. The dashboard provides a comprehensive view of key performance indicators such as total revenue, profit, profit margin, order trends, category performance, customer metrics, and regional sales distribution.

The project includes multiple analytical views such as Executive Sales Overview, Loss Analysis, and Customer & Sales Insights, allowing stakeholders to monitor business performance and identify improvement opportunities. Key findings highlight seasonal sales patterns, high-performing product categories, loss-generating products due to discounts, and the contribution of high-value customers.

Overall, this project demonstrates how data analytics and business intelligence tools can be used to analyze retail data, generate actionable insights, and support strategic decision-making.
